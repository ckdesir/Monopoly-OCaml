Module Player

module Player: sig .. end

Representation of dynamic player data.

type t

The abstract type of a player

type player_id = string

Type of player identifier

exception InsufficientFunds

Raised when a player tries to perform an action but has insufficient balance

val name : t -> player_id

name t is the string representing the name of the person playing as the player t

val piece : t -> string

piece t is the token that the player t is represented by on a physical monopoly board. For example, "thimble", "top hat" would be valid values of piece t on a traditional monopoly board

val current_square : t -> int

current_square t is the square position that player t occupies.

val balance : t -> int

balance t is the value of the mutable balance of player t

val properties : t -> Board.square_name list

properties t is the list of properties that are owned by the player represented by t

val bankrupt : t -> bool

bankrupt t represents whether or not the player t is bankrupt, and thus determines if they are still in the game

val make_bankrupt : t -> t

make_bankrupt t returns a copy of player t except now they are bankrupt, and thus out of the game

val sets : t -> (string * int) list

sets t returns the list of the sets owned by player t

val add_to_sets : t -> string * int -> t

add_to_sets t set returns an identical player as t except they now own the property set set

val add_to_properties : t -> Board.square_name -> t
val add_railroad : t -> t
val add_utility : t -> t
val railroads : t -> int
val utilities : t -> int
val get_set_by_name : t -> string -> string * int
val is_in_jail : t -> bool

is_in_jail t is whether or not the player represented by t is currently in jail.

val turns_in_jail : t -> int

turns_in_jail t returns the amount of consecutive turns the player t has spent in jail

val incr_turns_in_jail : t -> t

incr_turns_in_jail t returns a copy of the player t but having spent another turn in jail

val clear_turns_in_jail : t -> t

clear_turns_in_jail t returns a copy of the player t but having spent no turns in jail

val create : player_id -> string -> t

create n p instantiates a player of type t with name n, piece p, current_square of "GO", balance of 1500, properties [], is_bankrupt false, and get_out_of_jail_cards of 0.

val replace_a_set : t -> string -> string * int -> t
val pay : int -> t -> t -> unit

pay int t p alters two player's mutable balances to perform the physical action of paying in Monopoly. For example, if t had $600 and p had $400 then pay 100 t p would result in balance t being 500 and balance p being 500. Raises InsufficentFunds if t lacks balance to perform pay

val bank_transaction : int -> t -> unit

pay_bank amt t alters player t's balance by an amount amt. For example, when passing Go a player would see the equivalent of bank_transaction 200 t. Note: amt may be positive or negative, corresponding to receiving or paying money to the bank respectively. Raises InsufficientFunds if the player lacks funds.

val pass_go : t -> unit

pass_go t adds 200 to player t's balance, equivalent to bank_transaction 200 t

val jail_cards : t -> int

jail_cards t represents the amount of get-out-of-jail free cards that player t is currently holding. If jail_cards t is 0, then the player will be sent to jail if prompted

acquire t s returns an identical player as t except the square s is added t's list of properties

val trade : t ->
       t ->
       Board.square_name list -> Board.square_name list -> t * t

trade p1 p2 p1_props p2_props is a pair of players p1 and p2 with the various exchanged properties, mirroring if players traded squares in real life. Note: cannot trade get_out_of_jail cards.

val owns : t -> Board.square_name -> bool

owns t s returns the boolean of whether or not player t owns the square s

val incr_cards : t -> t

incr cards p returns the a player identical to p with one more get-out-of-jail-free card

val decr_cards : t -> t

incr cards p returns the a player identical to p with one less get-out-of-jail-free card

val trade_cards : t -> t -> int -> t * t

trade_cards p1 p2 amt is a pair of players p1 and p2 that have exchanged amt get-out-of-jail cards, with p1 giving amt to p2

val doubles : t -> int

doubles t returns the amount of consecutive doubles t has rolled

val set_position : t -> int -> t

set_position t x returns a new player object currently at position x

val incr_doubles : t -> unit

incr_doubles t increments the amount of consecutive doubles that t has

val clear_doubles : t -> unit

clear_doubles t sets the amount of consecutive doubles of player to 0

val change_jail_status : t -> t

change_jaiL_status t returns the player t that is now freed if t was in jail, or is now in jail if t was free

val remove_props : t -> Board.square_name list -> t
